## Regle d'utilisation

Chaque prompt doit produire des livrables exploitables: decisions, hypotheses, backlog, contrats API, evenements, modele de donnees, tests d'acceptation et risques. Si une information manque, l'agent doit proposer des options et poser les questions correspondantes.
